Gradle nightly

For snapshots we use our Artifactory snapshot repository. The recommended way to start using the snapshot: First, find the interesting distribution in the artifactory, then use it to configure the wrapper. Alternatively, you can manually download the snapshot distribution from the artifactory and install it locally.

Gradle Continuous Integration machines build distributions continuously. You can login as guest here to get the artifact list of the last successful build.
